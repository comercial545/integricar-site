Política de Privacidade — IntegriCar

Última atualização: 27 de abril de 2026

Dados coletados

O IntegriCar coleta nome, e-mail, dados de veículos, histórico de manutenções, agendamentos e imagens capturadas pela câmera para leitura de notas fiscais (OCR).

Uso dos dados

Os dados são utilizados exclusivamente para prestação dos serviços do IntegriCar. Não vendemos nem compartilhamos seus dados com terceiros para fins comerciais.

Câmera

As imagens capturadas pela câmera são usadas apenas para leitura de notas fiscais via OCR e não são armazenadas permanentemente.

Armazenamento

Os dados são armazenados de forma segura na plataforma Supabase. O usuário pode solicitar a exclusão de sua conta e dados pelo e-mail: contato@integricar.com
